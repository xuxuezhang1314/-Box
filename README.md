# ☁️ 云box (CloudBox)

> 极简可定制的二次元风格浏览器

![preview](preview.png)

## 特性

- 🎨 **极致视觉** - 黑白二次元风格，扫描线动画，故障艺术效果
- 🧩 **插件系统** - 支持用户编写 JavaScript 插件扩展功能
- 🎯 **极简交互** - 无冗余按钮，所有操作都有键盘快捷键
- 🏷️ **标签管理** - 底部导航栏，类似移动端的多标签体验
- 🖌️ **主题定制** - 支持导入自定义 CSS 文件

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘/Ctrl + T` | 新标签页（回到主页） |
| `⌘/Ctrl + W` | 关闭当前标签页 |
| `⌘/Ctrl + L` | 聚焦搜索框 |
| `⌘/Ctrl + ,` | 打开设置 |
| `⌘/Ctrl + ←` | 后退 |
| `⌘/Ctrl + →` | 前进 |
| `⌘/Ctrl + R` | 刷新 |
